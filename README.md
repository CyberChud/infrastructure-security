# infrastructure-security

A lab to host projects on infrastructure security using the following phases:

Phase 1 — Linux Hardening

Phase 2 — Network Segmentation

Phase 3 — Terraform and AWS

Phase 4 — Vulnerability Lifecycle

Phase 5 — CI/CD Security

Phase 6 — Identity and HPC
